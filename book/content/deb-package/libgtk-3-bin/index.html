libgtk-3-bin 套件探索

下載套件

下載「libgtk-3-bin」這個套件。

$ apt download libgtk-3-bin

解開套件

解開「libgtk-3-bin_3.18.9-1ubuntu3.1_amd64.deb」

$ dpkg -x libgtk-3-bin_3.18.9-1ubuntu3.1_amd64.deb libgtk-3-bin

$ tree libgtk-3-bin
